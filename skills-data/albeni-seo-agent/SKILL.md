## Cross-Skill Integration: The MT+PE Translation Pipeline

The SEO agent operates within a broader AI stack that includes three translation skills. Understanding how they connect is essential because content production for DE/FR/EN markets flows through this pipeline.

2. **Phase B (M4-M9):** Content matures, interlinking feeds PageRank from L2 pages, organic position climbs. Paid maintains but bids reduce as organic CTR grows.
3. **Phase C (M7-M12):** Organic reaches top 5. Paid scales down to brand defense only. Budget freed up shifts to the NEXT keyword in the transition queue.

When recommending budget allocation, always reference this transition queue and the current phase for each keyword. The goal is a rolling handoff: as each keyword transitions to organic, the freed budget funds the next one.
